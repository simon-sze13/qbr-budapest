# Prerequisites

## Before you join

- [ ] **Browser**: Current Chrome, Firefox, or Edge.
- [ ] **Network**: Access to the workshop tenant or demo environment (VPN only if your facilitator requires it).
- [ ] **Credentials**: Workshop user issued by facilitators—not your personal production admin unless directed.
- [ ] **Optional**: Terminal with `kubectl` or cloud CLI **only if** the lab script uses them (confirm with your facilitator).

## Accounts and roles

Your facilitator will confirm:

1. Which **organization / project** to use in Cloud Factory.
2. Whether you need **cloud credentials** or everything is pre-provisioned.
3. Any **IP allowlisting** or SSO steps.

!!! warning "Do not commit secrets"
    Never paste API keys, kubeconfigs, or passwords into this repository or public issues. Use your facilitator’s secret-handling process.

## Verify access

When your environment is ready, you should be able to:

1. Sign in to the **Cloud Factory** UI (URL provided in session).
2. See the **project** assigned for the workshop.
3. Open **Lab 1** and complete the first checklist item.
