# Glossary (workshop)

Short definitions for QBR conversations. For the full dictionary, see the **[Taikun glossary](https://docs.taikun.cloud/Glossary/Glossary/)**.

| Term | Meaning |
|------|---------|
| **Cloud Factory** | Cloudera’s unified experience for Kubernetes and related platform operations across clouds. |
| **Project** | A scoped workspace for resources, access, and policies—often maps to a team or workload boundary. |
| **Cluster** | A Kubernetes cluster registered or managed through the platform (provider-specific underneath). |
| **Catalog** | Curated applications or Helm charts available for governed deployment. |
| **Ingress** | Kubernetes API object (and controller) that routes HTTP/S to Services by host and path. |
| **LoadBalancer Service** | Service type that requests an external endpoint from the cloud or platform. |
| **CertManager** | Tooling to automate TLS certificate issuance and renewal (often paired with Ingress). |
| **Taikun Link** | Cloud Factory feature to expose a workload with a dedicated hostname and HTTPS with minimal manual DNS/TLS setup. |
| **Workshop tenant** | A sandbox environment used for training; not production data. |
